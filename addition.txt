==================================================
FEATURE 1 - Inventory Auto-Deduct
==================================================
Requirement Implemented:
- Added service-to-inventory mapping
- Added automatic stock deduction when an appointment becomes completed
- Added admin page to manage which products are consumed by each service

Functions Added / Updated:
- salonDeductInventoryForService(PDO $pdo, int $appointmentId)
  File: includes/salon_helpers.php
  Purpose:
  Deducts inventory based on the completed appointment's service, logs the deduction,
  and prevents duplicate deduction for the same appointment.

Files Added:
- admin/manage_service_inventory.php
  Purpose:
  Admin/receptionist page to assign inventory items and quantity used per service.

Files Modified:
- includes/salon_helpers.php
  Added:
  - service_inventory schema creation
  - inventory_deductions schema creation
  - salonDeductInventoryForService()
- appointments.php
  Added logic to:
  - call inventory deduction when appointment status becomes completed
- admin/dashboard.php
  Added quick action link to Service Inventory page
- database.sql
  Added tables:
  - service_inventory
  - inventory_deductions
  Added columns:
  - appointments.completed_at
  - appointments.inventory_deducted_at

FEATURE 2 - Automatic Purchase Orders
==================================================
Requirement Implemented:
- Auto-generate purchase orders when inventory reaches minimum stock or below
- Prevent duplicate open purchase orders for the same inventory item
- Trigger PO generation after inventory changes and after appointment completion
- Added admin notification for auto-generated purchase orders
- Added daily cron file

Functions Added / Updated:
- salonCheckAndGenerateAutoPO(PDO $pdo, ?int $createdBy = null)
  File: includes/salon_helpers.php
  Purpose:
  Runs auto-PO generation, checks for low stock, prevents duplicate coverage,
  and creates admin notifications.

- salonGenerateAutoPurchaseOrders(PDO $pdo, ?int $createdBy = null)
  File: includes/salon_helpers.php
  Purpose:
  Generates grouped purchase orders by supplier.

Files Added:
- cron/auto_purchase_orders.php
  Purpose:
  Daily cron entry point for automatic PO checking.

Files Modified:
- includes/salon_helpers.php
  Added:
  - salonCheckAndGenerateAutoPO()
  Updated:
  - salonGenerateAutoPurchaseOrders()
- inventory.php
  Added logic to:
  - trigger auto-PO check after add/edit/delete inventory actions
  - use POST + CSRF for delete action
- appointments.php
  Added logic to:
  - trigger auto-PO check after appointment completion
- database.sql
  Existing PO tables retained and integrated
